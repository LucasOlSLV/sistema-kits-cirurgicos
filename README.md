Sistema de Gestão de Kits Cirúrgicos
Projeto desenvolvido em Java com foco em Programação Orientada a Objetos (POO), inspirado na lógica de organização de kits cirúrgicos utilizada em ambiente hospitalar.

O sistema simula o gerenciamento de kits médicos, permitindo cadastro de cirurgias, montagem automática de itens com base no procedimento e atualização de status dos kits.

Funcionalidades:
Cadastro de kits cirúrgicos,
Montagem automática de itens por cirurgia,
Listagem de kits,
Atualização de status dos kits,
Estrutura baseada em Programação Orientada a Objetos.
 
Tecnologias utilizadas:
Java,
IntelliJ IDEA,
Programação Orientada a Objetos (POO),
ArrayList.

Exemplo de funcionamento
Ao cadastrar uma cirurgia do tipo "ombro", o sistema monta automaticamente um kit contendo:
Seringa
Luva
Anestesia
